Installation

How to install

Get started in minutes

G’MIC-Qt is a self-contained bundle — no MacPorts or Homebrew needed on your Mac.

1

Download & unzip

Click the download button above and unzip the file. You’ll get a folder named gmic_gimp_qt.

2

Copy to the GIMP plug-ins folder

Move gmic_gimp_qt to:

~/Library/Application Support/GIMP/3.0/plug-ins/

In Finder: Go → Go to Folder… and paste the path above.

3

Remove the quarantine flag

macOS marks downloaded files as quarantined. Open Terminal and run:

xattr -dr com.apple.quarantine "$HOME/Library/Application Support/GIMP/3.0/plug-ins/gmic_gimp_qt"
4

Restart GIMP

G’MIC will appear under Filters → G’MIC-Qt… in GIMP.

If G’MIC doesn’t appear, clear the plug-in cache and restart GIMP:
rm -f "$HOME/Library/Application Support/GIMP/3.0/pluginrc"
